高性能C++量化交易系统，支持多数据源、多策略和多交易所接入，采用模块化设计与可插拔架构。

## 快速开始

### 环境要求
- C++11及以上
- CMake 3.15+
- Boost 1.70+
- 操作系统：Linux (推荐Ubuntu 22.04+)



### 编译步骤

```shell
#### 克隆仓库
git clone https://github.com/example/quant-trading-system.git
cd QuantTrading/

#### 创建构建目录
mkdir build && cd build

#### 生成Makefile
cmake ..

#### 编译
make -j4
```
